# Payroll Manager

A command-line payroll management system built in Rust as part of my Stellar/Soroban learning path.

## Features
- Add employees across departments
- Deactivate employees
- Calculate total active payroll
- Filter employees by department
- Find the highest earner

## Concepts Practiced
- Structs and enums
- HashMap for data storage
- Option and Result error handling
- Iterators and closures
- Borrowing and references

## Running the Project
cargo run

## Project Structure
- `src/main.rs` — all logic and entry point
